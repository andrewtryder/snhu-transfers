
## 2024-07-03 - [Breadcrumb Navigation]
**Learning:** Structural/invisible SEO metadata (JSON-LD) is preferred over visual UI additions (visible breadcrumbs) for this codebase to maintain UI cleanliness while still getting rich snippet benefits.
**Action:** When adding SEO structured data, propose JSON-LD only first, unless visual UI changes are explicitly requested or obviously missing from standard templates.
